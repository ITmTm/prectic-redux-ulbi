## Функции
* Пополнение счета
* Снятие со счета
* Добавление клиента
* Получение клиента с базы данных

## Getting Started
Tools needed to run this app: `node` and `npm`

- `npm install` to install dependencies
- `npm start` to run the app
- Go to `http://localhost:3000`
